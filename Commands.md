# Comandos de git

## Comandos principales
```shell
git add . // Esto agrega un cambio al stack
git commit  // esto crea un paquete de cambios
git push -u origin <rama> // Esto crea la rama remota y publica los cambios
git checkout <nombre de rama> // Esto cambia de rama
git checkout -b "nombre de la nueva rama" // Esto crea y cambia a la nueva rama
git branch  // Esto muestra la rama actual
git status // Esto muestra el esatado de cambios
```
