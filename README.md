## Como clonar un repositorio

1. Ir al repositorio

2. Copiar la url o el formato http del repositorio

3. usar el comando de clonar `git clone <url>`

---

## Como iniciar un repositorio local

el comando para iniciar un repositorio local es `git init`


## Como agregar cambios

el comando para agergar cambios al track es `git add <archivo>/.`
